1. PSYCHOLOGIE (OPERATIV – NICHT THEORETISCH)
🔑 CORE WAHRHEIT
 
Menschen entscheiden NICHT logisch.
 
Sie entscheiden basierend auf:
 
Faktor​Bedeutung
Schmerz​Weg-von
Vorstellung​Hin-zu
Energie​Ease
Sicherheit​Trust
Timing​Momentum
🔴 DIE 5 PSYCHOLOGISCHEN BLOCKER
1. Unbewusstes Problem
 
👉 „Ist kein Thema“
 
→ braucht: Awareness / Reframe
 
2. Unklare Vorstellung
 
👉 „Klingt logisch“
 
→ braucht: Visualisierung / Konkretisierung
 
3. Überforderung
 
👉 „Keine Zeit“
 
→ braucht: Reduktion
 
4. Misstrauen
 
👉 „Klingt zu gut“
 
→ braucht: Sicherheit
 
5. Trägheit
 
👉 „Später“
 
→ braucht: Aktivierung
 
👉 Emma macht IMMER:
 
→ einen dieser 5 Punkte lösen
